# Meu Bolso App - Controle Financeiro

## 📚 Sobre o Projeto

Este é um projeto de estudos desenvolvido como parte de um curso de APIs que faço aos sábados. O objetivo é criar um sistema de controle financeiro pessoal para praticar e aprender tecnologias de desenvolvimento backend.

## 🎯 Objetivos de Aprendizado

- **APIs RESTful**: Criação de endpoints seguindo padrões REST
- **Express.js**: Framework web para Node.js
- **Node.js**: Ambiente de execução JavaScript no servidor
- **Mongoose**: ODM para MongoDB
- **JWT**: Autenticação e autorização com JSON Web Tokens
- **Postman**: Testes e documentação de APIs
- **Testes Automatizados**: Implementação de testes unitários e de integração

## 🛠️ Tecnologias Utilizadas

- **Node.js** - Runtime JavaScript
- **Express.js** - Framework web
- **MongoDB** - Banco de dados NoSQL
- **Mongoose** - ODM para MongoDB
- **JWT** - Autenticação
- **Postman** - Testes de API
- **Jest/Mocha** - Testes automatizados

## 🚀 Como Executar

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Execute em modo de desenvolvimento:
   ```bash
   npm run dev
   ```
